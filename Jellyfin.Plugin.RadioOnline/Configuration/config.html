Radio Online

Transmite audio a traves de Liquidsoap con programacion semanal de playlists. Requiere un contenedor Docker de Liquidsoap corriendo con el script radio.liq.

Configuracion Programacion Estado

Servidor Liquidsoap

Host del servidor Telnet de Liquidsoap. Si esta en el mismo servidor, usa "localhost".
Puerto del servidor Telnet de Liquidsoap (definido con set("server.telnet.port", ...) en radio.liq).

Mapeo de Rutas

Ruta raiz donde Jellyfin almacena los archivos multimedia en el host.
Ruta correspondiente dentro del contenedor Liquidsoap (definida en docker-compose volumes).

Configuracion de Jellyfin

Usuario con acceso a las bibliotecas de musica y playlists.
Al activar, el plugin enviara canciones a Liquidsoap durante las horas programadas.

Programacion Semanal

Asigna playlists a dias y horarios especificos. La transmision se activa solo durante las horas programadas. Si dos programas se solapan, el que empieza mas tarde tiene prioridad.
